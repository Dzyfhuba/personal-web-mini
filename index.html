Deskripsi

Saya adalah mahasiswa informatika yang sedang tertarik pada bidang pengembangan website. Dengan menggunakan bahasa pemrograman php dengan bantuan framework Laravel, Saya telah mengerjakan beberapa proyek website kecil-kecil, termasuk beberapa proyek yang saya kerjakan untuk tugas kuliah. Selain itu, saya sekarang mempelajari javascript untuk melengkapi keilmuan saya tentang pengembangan web.

Di samping pengembangan website, sekarang saya mempelajari sistem optimasi untuk menyelesaikan studi sarjana saya sehingga saya mempelajari python serta pelengkapnya. Saya juga pernah belajar Unity Engine untuk membuat game dan belajar modeling, rigging dan animating menggunakan Blender 3D.
